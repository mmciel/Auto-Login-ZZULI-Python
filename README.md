## 疑难解答

 - 脚本闪退或者程序闪退是为啥：
   - 以管理员方式运行
   - 先连接校园网（不用登录），在启动脚本或者程序
- 设置用户密码设置错了怎么办：
  - 找到程序同一目录下的login_data.ini文件删掉重启程序
  - 手动修改login_data.ini
- 脚本咋用：
  - 先安装python3
  - 安装pip包管理器
  - 安装 脚本中出现的模块
  - 双击运行
